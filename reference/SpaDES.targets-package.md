# SpaDES.targets: Orchestrate SpaDES Simulations with 'targets'

**\[experimental\]**

Run 'SpaDES' simulations as stages of a 'targets' pipeline, passing the
discrete components each stage needs (plain objects as ordinary targets,
spatial objects as `format = "file"` targets) rather than whole
simLists.
